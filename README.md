# Outskill Claude Expert Resources

This repository houses the **Outskill Claude Expert** skill for Claude Code, along with a curated library of resources, playbooks, and architectural guides from the Outskill Masterclasses.

## 🧠 The Skill  
The `outskill-claude-expert` skill acts as an advanced AI automation architect. It bridges Outskill methodologies, Snarktank PRD frameworks, and Model Context Protocol (MCP) integrations to guide Claude Code in prioritizing clarity, spec-driven development, and modular agent architecture.

### Installation  
You can install this skill directly into Claude Code using the `skills.sh` registry:  
`npx skills add <your-github-username>/outskill-claude-resources --skill outskill-claude-expert`

## 📚 Outskill Documentation & Resources  
Inside the `docs/` folder, you will find supplementary materials to master Claude Code and Agentic workflows:  

* **`claude logic flows.pdf`:** Diagrams and logic flows for building multi-tenant SaaS agent architectures.  
* **`curated-links.md`:** A master resource hub with curated links to essential MCPs, Substacks, and GitHub frameworks (e.g., n8n, Compound Engineering).  
* **`50% context rules.pdf`:** Deep dive into `memory.md`, `CLAUDE.md`, and the 50% context limit rule to prevent context rot.  
* **`terminal foundations.pdf`:** Fundamental guides on terminal basics, navigation, and environment variables.
* **Snarktank PRD Toolkit:** Includes `create-prd.md`, `generate-tasks.md`, and `process.md` to aid in spec-driven development mapping.
* **`claudecode_transcripts/`:** Folder containing the full raw session transcripts for deep-dive learning.

## ⚙️ Usage  
When starting a new project, use this skill to generate your `it_process.md` (PRD) and `CLAUDE.md` files before writing any code. Let the skill help you map out your "intern" agents and select the right MCP tools.
